# 发射器后台骨架

## 当前已经具备

- 北京时间07:40自动生成草稿；
- 北京时间08:00发布；
- 通过OpenAI Responses API进行联网搜索与生成；
- 对3条宏观、3—5条快讯、5个名词进行结构检查；
- 生成失败时保留错误状态，不用旧记忆冒充今日新闻；
- 提供健康状态与最新晨报API；
- 默认只监听本机，避免无意公开。

## 接口

- `GET /api/health`：查看发射器状态；
- `GET /api/reports/latest`：读取最新晨报JSON；
- `POST /api/generate`：手动触发一次生成。

## 尚未接通

正式联网需要配置 `OPENAI_API_KEY`。线上阅读还需要将这套服务部署到个人云环境，并让前端读取 `/api/reports/latest`。

API密钥只能放在云端环境变量里，不能写进网页或微信小程序。
